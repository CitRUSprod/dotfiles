---
description: Ревьюит открытый PR через @code-reviewer и публикует комментарий
---

Текущая ветка: !`git branch --show-current`

Шаги:

1. **Определи PR:**
   - Если передан аргумент (`$ARGUMENTS`) — используй его как номер PR
   - Иначе получи PR для текущей ветки: `gh pr view --json number,url,headRefName,baseRefName,title`
   - Если PR не найден — сообщи пользователю и прекрати выполнение

2. **Получи изменения:**
   - Выполни `gh pr diff <number>` и передай результат агенту `@code-reviewer`

3. **Запусти ревью:**
   - Попроси `@code-reviewer` проанализировать diff
   - Дождись результата

4. **Опубликуй результат:**
   - Выполни `gh pr comment <number> --body "<результат ревью>"`
   - Сообщи пользователю ссылку на PR и что ревью опубликовано
